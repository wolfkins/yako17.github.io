Hololive Commands

View All Streams

Usage

$$hololive

Description

View live and upcoming streams for all Hololive VTubers. Data is updated every 5 minutes.

Note

The alias $$hl works as well.

View Specific VTuber

Usage

$$hololive VTuber

Description

View data specific to the input VTuber, such as their status, viewer count, etc.

View Holostars Data

Usage

$$hololive holostars

Description

View all data for the Holostars section of Hololive.

Hololive Notification Settings

Usage

$$hlnotify

Description

View the server's settings for the Hololive notifier.

Hololive Notifier Setup

Usage

$$hlnotify setup channel

Description

Setup Hololive notifications in the given channel.

Required Permissions

Manage Channels

Hololive Notifier Message

Usage

$$hlnotify setmsg

Description

Opens a prompt to set a custom message that displays when a VTuber goes live.

Required Permissions

Manage Channels

Note

The text [vtuberName] and [videoTitle] can be used to insert information about the stream into the message.


Example: [vtuberName] has gone live! => Usada Pekora has gone live!

Add To Notifier

Usage

$$hlnotify add VTuber

Description

Adds the specified VTuber to the notifier. Notifications will be recieved whenever they go live.

Required Permissions

Manage Channels

Remove From Notifier

Usage

$$hlnotify remove VTuber

Description

Removes the specified VTuber from the notifier.

Required Permissions

Manage Channels

Preview Notifier Message

Usage

$$hlnotify preview

Description

Preview the message that gets displayed whenever a VTuber goes live.

Required Permissions

Manage Channels

Personal Notifiers

Donators get access to personal Hololive notifiers sent to their Direct Messages. See the Donator page for more information.
