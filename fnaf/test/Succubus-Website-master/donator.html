Donator Commands

Song Lyrics

Usage

lyrics song title

Description

Search the Genius lyrics database for the input song lyrics.

Extra Playlists

Description

Tier 1 Donators get 6 Playlists with 30 Songs Each!
Tier 2 Donators get 12 Playlists with 50 songs each!

Server Premium

Usage

premium

Description

Marks the server you are currently in as a Premium servers, and gives it access to special perks.

Note

Tier 2 Donator is required to give servers Premium status

Warning

Once you give a server premium, you CANNOT change it. Make your choice count!

Hololive Personal Notification Settings

Usage

$$hlnotifyme

Description

View your personal settings for the Hololive notifier.

Personal Notifier Setup

Usage

$$hlnotifyme setup

Description

Setup Hololive notifications in your Direct Messages.

Personal Notifier Message

Usage

$$hlnotifyme setmsg

Description

Opens a prompt to set a custom message that displays when a VTuber goes live.

Note

The text [vtuberName] and [videoTitle] can be used to insert information about the stream into the message.


Example: [vtuberName] has gone live! => Usada Pekora has gone live!

Add To Personal Notifier

Usage

$$hlnotifyme add VTuber

Description

Adds the specified VTuber to your notifier. Notifications will be recieved whenever they go live.

Note

Tier 1 Donators get access to 3 personal notifiers, while Tier 2 Donators get access to 6

Remove From Personal Notifier

Usage

$$hlnotifyme remove VTuber

Description

Removes the specified VTuber from your notifier.

Preview Personal Notifier Message

Usage

$$hlnotifyme preview

Description

Preview the message that gets displayed in your DMs whenever a VTuber goes live.
