SauceNAO Commands

Sauce An Image

Usage

$$sauce [url]

Description

Search the SauceNAO databases for the given image url. If no image url is provided, the first image in the past 20 messages will be used.

Note

Valid image formats include .PNG, .JPG, and .GIF

SauceNAO Settings

Usage

$$saucenao

Description

View the current SauceNAO settings for the server.

Toggle Input Deletion

Usage

$$saucenao toggledel

Description

Toggle whether input images should be deleted or not. (Only when a URL is given)

Required Permissions

Manage Messages

Set Result Count

Usage

$$saucenao setresults amount

Description

Modify the amount of results the sauce command returns, from 1 to 5.

Required Permissions

Manage Messages

Lock Sauce Command

Usage

$$saucenao lock channel

Description

Locks the sauce command to the specified channel, so that sauce queries may only be run in that channel.

Required Permissions

Manage Channels
